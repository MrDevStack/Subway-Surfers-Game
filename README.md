# Subway Surfers Game

A fun endless runner game inspired by Subway Surfers. Run on railway tracks, dodge trains, collect coins, and try to beat your high score.

## Features

- Endless runner gameplay
- Coin collection system
- Train obstacles
- Smooth player controls
- Score and high score system
- Colorful 3D environment
- Mobile-friendly gameplay

## How to Play

- Swipe left or right to change lanes
- Swipe up to jump
- Swipe down to slide
- Avoid trains and obstacles
- Collect as many coins as possible
